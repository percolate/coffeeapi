Coffee App Instructions

Use the API endpoint https://coffeeapi.percolate.com/api/coffee/?api_key=api_key to get the data for the coffee list.

Implement an application that follows the design storyboard found here. Our logo can be found here and our color palette is #F16421 Orange, #AAAAAA Gray, and #666666 Gray-Dark.

Android Requirements

  • Your application must be built using Android Studio.
  • Target Android 4.0 and up (API 14).

iOS Requirements

  • Do all layout using autolayout in code.
  • Target iOS 7.1+

You can make your own assumptions about any details not mentioned. Feel free to email us with any questions and be sure to state your assumptions as part of your submission.

To submit your application, please provide us access to a git repo.
